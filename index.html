TNYT React App

See Live View Code

A fully responsive react CRUD application using The New York Times' news API with Supabase for Database, Auth and Hosting. This app pulls headlines from all the main sections of The New York Times. Users are able to login and leave comments on all articles.

This app features:
- Various React hooks such as useEffect, useHistory and useLocation to make API calls and define routes.
- Multiple API calls to fetch News and Weather data.
- State stored and persisted using Redux and local storage.
- User authentication to allow for article comment creation, deletion and editing.

I worked with The New York Times' style guide to retain brand consitency. MaterializeUI was used for the icons.
